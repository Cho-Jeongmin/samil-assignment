# 삼일PwC엑셀러레이션센터 2026년 하반기 프론트엔드 개발자 채용 과제

선택 과제는 수행하지 않았습니다.

## 🍋 실행하기
1. 프로젝트 클론하기
```
git clone https://github.com/Cho-Jeongmin/samil-assignment.git
```

2. 프로젝트 폴더로 이동하기
```
cd samil-assignment
```

3. 패키지 설치하기
```
npm install
```

4. .env 파일에 백엔드 주소 작성하기
<br>
NEXT_PUBLIC_API_BASE_URL=백엔드api주소

6. 개발 서버 실행하기
```bash
npm run dev
```

6. 브라우저에서 [http://localhost:3000](http://localhost:3000) 접속하기
